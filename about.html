About The Server

This server is meant to be a fun minecraft server and social expiriment. This server sees what happens when we introduce fake Ecenomics, and Politics into a minecraft server. We keep everything fake and do not bring real politics or issues into the game, this is strictly to have fun.

History

In Season 1 we started with 2 parties for the first month, using a slideshow for the election results. The Creeper Party won over the Steve Party 7-4.

We almost had a second election with a third party entering known as the Wolves. But the server ended before that could fully start

Mods and Plugins

We have plenty of Quality of Life Plugins and Mods.

Home Commands

  • /sethome - Sets your current cords as "Home"
  • /home - Sends you to "Home"
  • /deletehome - Deletes your "Home" cords

Warps

  • /warp set [name] - Sets current cords as a warp
  • /warp [name] [password if set] - Warps you to a set spot
  • /warp delete [name] - Deletes that warp (Only made by you)
  • /warp password [name] [password] - Sets a password to that warp (Only made by you)
  • /warp password [name] clear - Removes password from that warp (Only made by you)
  • /warp list - Shows list of Warps
  • /warp update [name] - Updates cords of that warp

TPA

  • /tpa [username] - Sends Request to Teleport to that player
  • /tpaccept [name to specify if needed] - Accepts most recent Teleport Request
  • /tpacancel - Cancels a Teleport Request to someone
  • /tpatoggle - Toggles if people can request to teleport to you
